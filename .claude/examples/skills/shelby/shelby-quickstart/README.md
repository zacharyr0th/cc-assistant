# Shelby Protocol Quickstart Skill

A comprehensive onboarding skill for developers new to the Shelby Protocol, guiding through initial setup, token acquisition, and first uploads.

## What This Skill Does

This skill provides step-by-step guidance for:
- Understanding Shelby Protocol basics
- Choosing between CLI, SDK, and media player
- Setting up accounts and funding with tokens
- Completing first file upload
- Next steps and learning resources

## When It's Invoked

The skill activates when users:
- Ask "how do I get started with Shelby?"
- Mention "Shelby setup" or "new to Shelby"
- Need help choosing Shelby tools
- Want to understand token requirements
- Need first-time setup guidance

## What You'll Learn

- Shelby Protocol overview and use cases
- Token system (Aptos + ShelbyUSD)
- CLI quickstart (most common path)
- SDK quickstart (for developers)
- Media player quickstart (for React apps)
- Next steps and resources

## Quick Example

**User**: "I'm new to Shelby. How do I start?"

**Skill Response**: Guides through CLI installation, token funding, and first upload with verification steps.
